# 유스케이스 기반 이벤트 도출 (도메인 관점)

## 기준
- 이벤트는 **어그리거트 범위를 벗어난 행동**이 필요하거나 **현재 도메인과 관련되지 않은 부가기능**이 필요한 경우로 한정한다.
- 아래 항목은 코드 유무와 무관하게 **유스케이스 흐름에서 발생 가능한 이벤트 후보**를 도출한 것이다.

## 모듈별 이벤트 후보
